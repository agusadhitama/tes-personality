# 🧠 Tes Personality

**Tes Personality** adalah aplikasi web sederhana yang digunakan untuk mengetahui tipe kepribadian seseorang melalui serangkaian pertanyaan.

Pengguna akan menjawab beberapa pertanyaan, kemudian sistem akan menganalisis jawaban tersebut untuk memberikan hasil tipe kepribadian.

---

## 🚀 Live Demo

🌐 https://agusadhitama.github.io/tes-personality/

---

## 📌 Deskripsi

Aplikasi ini dirancang untuk memberikan gambaran mengenai karakter atau kecenderungan perilaku seseorang melalui kuis interaktif.

Pengguna hanya perlu menjawab beberapa pertanyaan yang tersedia, kemudian sistem akan menghitung skor dan menampilkan hasil kepribadian.

Tes seperti ini sering digunakan untuk membantu seseorang **lebih memahami dirinya sendiri, gaya komunikasi, dan cara mengambil keputusan**.

---

## ⚙️ Cara Menjalankan Project

Clone repository:

```
git clone https://github.com/agusadhitama/tes-personality.git
```

Masuk ke folder project:

```
cd tes-personality
```

Lalu buka file:

```
index.html
```

di browser.
